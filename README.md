# TorchX


For the latest documentation, please refer to our [website](https://pytorch.org/torchx).


## Requirements
TorchX SDK (torchx):
* python3 (3.8+)
* torch

TorchX Kubeflow Pipelines Support (torchx-kfp):
* torchx
* kfp

## Installation
```bash
# install torchx sdk
pip install torchx

# install torchx kubeflow pipelines (kfp) support
pip intall torchx-kfp
```
